# Hello-Gemini
Hello Gemini is a voice-activated application designed to interpret user commands and execute corresponding functions using a Gemini model.
The Gemini model processes voice input, determines the user's intention, and triggers the appropriate action from a predefined list of functions.

## Functionality
- Voice Input: Users interact with Hello Gemini by speaking voice commands.
- Gemini Model: Processes voice input to understand user intent.
- Function Execution: Determines the desired function from a list based on user input.
- Supported Functions: Currently, the supported function is "open notepad".
